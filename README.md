一个基于 Tkinter 的 JSON节点路径查询可视化工具，实现了文件选择、多节点查询和结果展示功能。

工具特点和优化点：
可视化界面：使用 Tkinter 创建了直观的图形界面，包括文件选择、节点输入和结果展示区域

功能完善：
1. 支持通过文件浏览器选择 JSON 文件
2. 支持输入多个目标节点（逗号分隔）
3. 显示节点完整路径及对应的值
4.对数组类型添加索引标识（如[0]），使路径更清晰

用户体验优化：
1. 支持结果清空和错误提示
2. 界面可调整大小
3. 输入区域支持自动扩展
4. 结果区域使用带滚动条的文本框，方便查看大量结果

代码结构优化：
- 使用类封装，结构更清晰
- 增加异常处理，提高稳定性
- 支持节点输入时的空格自动忽略

使用方法：
1. 运行程序后，点击 "浏览" 按钮选择要解析的 JSON 文件
2. 在节点输入框中输入要查询的节点名称（多个节点用逗号分隔）
3. 点击 "查询节点路径" 按钮，结果会显示在下方的文本区域
4. 可点击 "清空结果" 按钮清除当前结果
